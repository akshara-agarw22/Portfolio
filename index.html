MY PROJECTS

BMI Calculator

A simple yet effective Body Mass Index(BMI) Calculator built using HTML,CSS and JavaScript.The tool allows users to input their height and weight to instantly calculate their BMI and receive health-related feedback based on standard BMI categories.It features a beautiful, clean and responsive design and real-time validation for a smooth user-experience.

Digital Clock

A real-time digital clock developed using HTML,CSS and JavaScript. It displays the current hours,minutes and seconds in a sleek foramt and updates every second.The clock adapts the system time and is styled for both desktop devices.

Guess The Number Game

An interactive number guessing game where the player tries to guess a randomly generated number between the range 1 to 100.Built using HTML,CSS and JavaScript , the game provides feedback on whether the guess is too high or too low and tracks the score,encouraging repeated play and logic-based guessing.

Selective Color Changer

A fun and interactive project that allows users to change the backgorund color on the webpage by selecting the desried color.Using HTML,CSS and JavaScript and event listeners, it demonstrates dynamic DOM manipulation and user interaction with buttons.

Unlimited Color Changer

A dynamic web tool that continuously changes the background color on presiing the start button and finally stops on pressing the stop button at random intervals.Created using HTML,CSS and JavaScript timers and color generating logic,this project showcases the use of setInterval,DOM styling , and user control over animations and color effects.
